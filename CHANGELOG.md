# Changelog

All notable changes to WebP Forge will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-07

### Added
- 🖼️ Drag & drop UI for bulk image conversion (up to 100 files)
- 📱 HEIC/HEIF support for iPhone photos
- 🔁 Lossless and lossy compression options
- 📐 Smart resize with aspect ratio preservation
- 👁️ Live before/after preview with size comparison
- 📦 Batch ZIP download for converted files
- 📡 Full REST API with endpoints for conversion, download, and preview
- 🧹 Auto cleanup system (files removed after 1 hour)
- ⚡ Sharp/libvips engine for fast processing
- 🔒 100% self-hosted, privacy-focused architecture
- 📚 Comprehensive API documentation
- 🎨 Modern, responsive UI design

### Supported Formats
- Input: JPG, PNG, GIF, BMP, TIFF, DNG, HEIC, HEIF
- Output: WebP (lossless and lossy)
