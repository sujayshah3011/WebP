REST API
Use the conversion engine programmatically — perfect for bulk pipelines, CI/CD, or automation.
POST /api/convert Convert one or many images
Field Type Description
images file[] One or multiple image files (multipart/form-data)
quality number WebP quality 1–100. Default: 82
lossless "true"/"false" Enable lossless compression. Default: false
effort number Compression effort 0–6. Default: 4
resize JSON string {"width":1920,"height":1080,"fit":"inside"}
curl -X POST http://localhost:3000/api/convert \ -F "images=@photo.jpg" \ -F "quality=85" \ -F "effort=5"
# Bulk — send multiple files at once curl -X POST http://localhost:3000/api/convert \ -F "images=@img1.jpg" \ -F "images=@img2.png" \ -F "images=@img3.jpg" \ -F "quality=82" \ -F "resize={\"width\":1920}"
// JavaScript / Node.js const form = new FormData(); files.forEach(f => form.append('images', f)); form.append('quality', '85'); form.append('lossless', 'false'); const res = await fetch('http://localhost:3000/api/convert', { method: 'POST', body: form }); const data = await res.json(); // data.results[].downloadUrl → use to download each file console.log(`Saved ${data.totalSavingsPercent}% overall`);

curl -O http://localhost:3000/api/download/converted-abc123.webp

curl -O "http://localhost:3000/api/download-zip?files=a.webp,b.webp,c.webp"
